В этом репозитории лежат проекты, которые я выполнила во время учёбы в "Яндекс Практикуме".

Внутри каждого проекта есть короткое описание, цели и пошаговая работа, которую я провела, чтобы достичь целей. По этому можно понять, какими инструментами я владею и как рассуждаю.

Список проектов:

1. Preprocessing data for analysis for online-game

  Я выполняю предобработку данных в датасете до начала анализа: выявляю явные и неявные дубликаты, удаляю их, нахожу и обрабатываю пропуски, привожу данные к единообразию, а также отбираю нужный срез данных и выделяю определённые категории данных, которые нужны для анализа. Все эти шаги - обязательная база для анализа.

2. Creation and analysis of charts for a financial company

  Создаю разные типы графиков с помощью библиотеки matplotlib и анализирую их, чтобы дать финансовой компании представление о ситуации с инвестированием стартапов на рынке.

3. Calculating metrics in SQL and interpreting results for Yandex Books

  С помощью SQL я рассчитываю метрики сервиса "Яндекс Книги" за осень 2024 года: MAU авторов, MAU произведений, Retention Rate за каждый день в течение 7 дней после установки, LTV по Москве и Санкт-Петербургу и среднюю выручку за прослушанный/прочитанный час - аналог среднего чека. Также я интерпретирую результаты расчётов.

4. Analysis of AB testing results for online-shop

  Здесь я контролирую A/B-тестирование: проверяю корректность его проведения, выясняю его результат и оцениваю его валидность с помощью статистического теста, а также даю рекомендации по дальнейшим шагам, чтобы улучшить проверяемую метрику.
