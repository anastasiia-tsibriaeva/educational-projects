В этом репозитории лежат проекты, которые я выполнила во время учёбы в "Яндекс Практикуме". 

Внутри каждого проекта есть короткое описание, цели и пошаговая работа, которую я провела, чтобы достичь целей. По этому можно понять, какими инструментами я владею и как рассуждаю.

Список проектов:
- Preprocessing data for analysis

  Подробно показывает, как я выполняю предобработку данных в датасете до начала анализа: выявляю явные и неявные дубликаты, удаляю их, нахожу и обрабатываю пропуски, привожу данные к единообразию, а также отбираю нужный срез данных и выделяю определённые категории данных, которые нужны для анализа. Все эти шаги - обязательная база для анализа.

- Creation and analysis of charts for a financial company

  Показывает, как я создаю разные типы графиков с помощью библиотеки matplotlib и анализирую их, чтобы дать финансовой компании представление о ситуации с инвестированием стартапов на рынке.
